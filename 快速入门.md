# 初始化一个 git 仓库
进入新建的仓库文件夹（待初始化的仓库）。
```
git init 
git add <filename>
# git add .
git remote add <name> <git-ssh>
#<git-ssh> example: git@github.com:cx-333/learngit.git commit -m '<commit message>'
# first push to github
git push -u <name> master
# later push
git push <name> master
# 查看仓库当前状态
git status
```
